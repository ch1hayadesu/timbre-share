# 音色共享平台 — 数据字典

| 项目 | 内容 |
|------|------|
| 数据库 | MySQL 8.4 |
| 引擎 | InnoDB |
| 字符集 | utf8mb4 |
| 排序规则 | utf8mb4_unicode_ci |
| 表数量 | 8 |
| 索引数量 | 24 |
| 外键数量 | 10 |
| 预设端口 | 3306 |
| 绑定地址 | 127.0.0.1 |

---

## 数据库账户

| 用户名 | 用途 | 可连主机 | 权限范围 |
|--------|------|----------|----------|
| `app_user` | Java Spring Boot 应用 | 127.0.0.1, localhost | timbre_share: SELECT, INSERT, UPDATE, DELETE |
| `readonly_user` | 报表/监控 | %, 127.0.0.1, localhost | timbre_share: SELECT |
| `admin_user` | DBA 运维 | 127.0.0.1, localhost | timbre_share: ALL PRIVILEGES WITH GRANT OPTION |

认证插件: `caching_sha2_password`（MySQL 8.4 默认）

## 当前数据量（测试环境）

| 表名 | 行数 | 说明 |
|------|:----:|------|
| user | 5 | 1管理员(seed) + 3正常测试 + 1边界测试 |
| voice | 16 | 8预设音色(seed) + 3克隆正常 + 2边界 + 3异常(含失败/处理中) |
| tts_record | 11 | 3正常 + 6边界(极值语速/音量/音调) + 2异常(空文本/处理中) |
| script_dub_task | 3 | 1正常 + 1边界(超长剧本) + 1异常(失败任务) |
| voice_share | 12 | 8预设分享(seed) + 2正常 + 1边界 + 1异常(已下架) |
| voice_download | 3 | 3条正常下载记录 |
| verification_code | 3 | 1正常(未过期) + 1已过期 + 1已使用 |
| notification | 3 | 1系统模板(seed) + 2业务通知(克隆完成/配音完成) |

## 1. 用户表 (`user`)

| # | 字段 | 类型 | 必填 | 默认值 | 说明 |
|:-:|------|------|:---:|:------:|------|
| 1 | user_id | BIGINT | Y | AUTO_INCREMENT | 主键，用户ID |
| 2 | phone | VARCHAR(20) | Y | — | 手机号，唯一索引 |
| 3 | membership_level | TINYINT | Y | 0 | 会员等级 0:普通 1:高级(MVP预留) |
| 4 | created_at | DATETIME | Y | CURRENT_TIMESTAMP | 创建时间 |
| 5 | updated_at | DATETIME | Y | ON UPDATE CURRENT_TIMESTAMP | 更新时间，行更新时自动刷新 |

**索引：** `idx_user_phone`(UNIQUE) · `idx_user_created_at`
**对应REQ：** REQ-001
**对应SQL：** `sql/02_seed_data.sql` — 插入管理员(user_id=0，system) · `sql/04_test_data.sql` — +4用户
