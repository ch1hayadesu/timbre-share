## 一、设计理念

### 1.1 核心关键词

- **专业可信赖**：企业级产品的稳定感与专业性
- **AI 智能化**：通过微妙的渐变色、玻璃态效果传递 AI 技术感
- **清晰高效**：信息层级明确，操作路径最短化
- **创作友好**：为创作者提供沉浸、流畅的使用体验
- **声音可视化**：通过波形、频谱等视觉元素强化「音色」品牌识别

### 1.2 设计原则

1. **减少认知负荷**：每个页面聚焦单一核心任务，复杂操作用步骤条引导
2. **即时反馈**：所有异步操作（上传、克隆、合成、配音）必须有明确的 loading 状态和进度指示
3. **容错设计**：异常状态有清晰的中文提示和恢复入口
4. **渐进式展示**：核心功能直接可达，高级功能折叠收纳
5. **声音即品牌**：将音频波形、频谱图作为核心视觉符号贯穿全平台

## 二、色彩体系

### 2.1 主色调

音色共享平台以「声音」为核心，采用**声波紫**（Violet Wave）作为品牌主色，象征声音的频率与创造性，区别于传统蓝/绿色企业产品。

```
品牌主色 (Brand Primary)：
- 声波紫 Primary-6:  #7C3AED  ← 核心品牌色（按钮、图标、高亮）
- 深紫 Primary-7:     #6D28D9  ← hover 状态
- 浅紫 Primary-5:     #8B5CF6  ← 浅色背景、选中态底色
- 极浅紫 Primary-1:   #F5F3FF  ← 页面级浅底色块

功能色：
- 链接蓝 Link:        #3B82F6  ← 超链接、文字按钮
- 成功绿 Success:     #10B981  ← 操作成功、已就绪状态
- 警告橙 Warning:     #F59E0B  ← 警告提示、处理中状态
- 错误红 Error:       #EF4444  ← 错误提示、失败状态
- 信息蓝 Info:        #6366F1  ← 信息提示、引导类提示
```

### 2.2 中性色

```
文字色：
- 标题文字 Text-Primary:   #1F2937  (gray-800) ← 页面标题、卡片标题
- 正文文字 Text-Body:      #374151  (gray-700) ← 正文、表单标签
- 次要文字 Text-Secondary: #6B7280  (gray-500) ← 辅助说明、时间戳
- 禁用文字 Text-Disabled:  #9CA3AF  (gray-400) ← 禁用态文字

- 深分割 Border-Dark:      #D1D5DB  (gray-300) ← 强调分割线
```

### 2.3 渐变色彩

用于 AI 相关模块和品牌强化场景：

```
AI 渐变 (Gradient-AI)：
  linear-gradient(135deg, #7C3AED 0%, #3B82F6 100%)
  用途：AI 处理中的动效、AI 标识徽章、数据大屏

音频渐变 (Gradient-Audio)：
  linear-gradient(90deg, #8B5CF6 0%, #A78BFA 50%, #C4B5FD 100%)
  用途：音频波形可视化、播放进度条

卡片渐变 (Gradient-Card)：
  linear-gradient(180deg, #F5F3FF 0%, #FFFFFF 100%)
  用途：精选音色卡片、推荐位背景
```

## 三、排版层级


### 3.2 字体大小与行高

```
层级    字号 (px)   行高    字重          用途
─────────────────────────────────────────────────────
H1      30          1.3     Bold (700)    页面大标题（如登录页、空状态引导）

Body-S  13          1.55    Regular(400)   辅助说明、时间戳、标签文字
Caption 12          1.5     Regular(400)   脚注、数据单位、徽章文字
```

### 3.3 排版规则

| 规则 | 说明 |
|------|------|
| **标题间距** | H1 与内容间距 24px，H2 与内容间距 20px，H3 与内容间距 16px |
| **段落间距** | 段落之间间距 12px（Body 行高的 0.5 倍） |
| **文字截断** | 单行超出使用省略号（`text-overflow: ellipsis`），最多两行 |
| **英文数字** | 中英文混排时，英文/数字前后加 1 个半角空格（或使用自动间距） |
| **对齐方式** | 标题左对齐，表格数字右对齐，状态标签居中对齐 |
| **按钮文字** | 按钮文字统一 14px Medium (500)，中文 2-4 字最佳，不超过 6 字 |

## 八、动效规范

### 8.1 过渡动画

```
持续时间 Token:
  duration-fast:    150ms  ← 微交互（hover、focus）
  duration-base:    250ms  ← 标准过渡（显示/隐藏、切换）
  duration-slow:    400ms  ← 复杂动画（弹窗、抽屉、页面切换）

缓动函数:
  ease-out:   cubic-bezier(0.00, 0.00, 0.20, 1.00)  ← 元素出现
  ease-in:    cubic-bezier(0.40, 0.00, 1.00, 1.00)  ← 元素消失
  ease-in-out: cubic-bezier(0.40, 0.00, 0.20, 1.00) ← 来回动画
  spring:     cubic-bezier(0.34, 1.56, 0.64, 1.00)  ← 弹性效果
```

### 8.2 具体动效定义

| 场景 | 动效 | 时长 | 缓动 |
|------|------|------|------|
| 按钮 hover | 背景色渐变 | 150ms | ease-out |
| 按钮 active | scale(0.98) | 100ms | ease-out |
| 卡片 hover | translateY(-2px) + shadow | 250ms | ease-out |
| 弹窗打开 | scale(0.95→1) + opacity(0→1) | 250ms | spring |
| 弹窗关闭 | scale(1→0.95) + opacity(1→0) | 200ms | ease-in |
| 抽屉滑入 | translateX(100%→0) | 300ms | ease-out |
| 下拉菜单 | opacity(0→1) + translateY(-8→0) | 200ms | ease-out |
| Toast 提示 | translateY(-16→0) + opacity(0→1) | 300ms | spring |
| 页面切换 | opacity(0→1) + translateY(8→0) | 300ms | ease-out |
| 列表项进入 | stagger: 50ms/项, opacity + translateY | 300ms | ease-out |
| 进度条 | width 过渡 | 400ms | ease-out |
| 脉冲动画 | scale(1→1.05→1) | 1500ms | ease-in-out, 循环 |
| 骨架屏 | shimmer 渐变移动 | 1500ms | ease-in-out, 循环 |
| 音频波形 | 条状高度随机变化 | 300ms | ease-in-out, 循环 |

### 8.3 AI 处理专属动效

```
AI 思考动画：
  - 三个圆点依次闪烁 (bounce)，间距 200ms
  - 配合文字「AI 正在处理中...」

音频波形动画：
  - 多根竖条高度在 20%-100% 之间随机变化
  - 过渡时间 300ms，模拟真实音频波形

克隆进度动效：
  - 步骤条当前步骤脉冲
  - 进度条使用 Gradient-AI 渐变 + shimmer 效果
```

```
  - 最近音色：横向滚动的音色卡片列表（最多 6 个）
  - 最近合成：最近 5 条合成历史记录

特殊处理：
  - 新用户：无音色时展示引导卡片「开始你的第一次音色克隆」
  - 加载态：骨架屏（统计数字闪烁 + 卡片占位）
```

### 11.2 音色管理页

```
页面目标：管理用户所有音色

布局：
  - 顶部工具栏：搜索框 + 筛选（来源：全部/克隆/下载/预设）+ 状态筛选 + [克隆新音色] 按钮
  - 音色卡片网格（4列→3列→2列→1列 响应式）
  - 每张卡片：波形可视化区 + 音色名 + 来源标签 + 状态标签 + 操作（试听/分享/删除）

操作：
  - 点击卡片 → 音色详情抽屉（右侧滑出）
  - 详情抽屉：音色信息 + 试听播放器 + 操作按钮组
  - 删除：二次确认弹窗
  - 分享：状态为「就绪」才可操作
```

### 11.3 音色克隆页

```
页面目标：引导用户完成音色克隆

布局：
  - 步骤条：上传音频 → 预处理 → 克隆中 → 完成
  - 步骤 1（上传区）：拖拽上传区 + 格式提示 + 文件信息
  - 步骤 2（自动进行）：预处理进度
  - 步骤 3（克隆中）：进度条 + 预计时间 + 取消按钮
  - 步骤 4（完成）：试听播放器 + 音色命名 + 保存按钮

特殊状态：
  - 上传音频过短/无人声：错误提示 + 重新上传引导
  - 克隆失败：失败原因 + 重试按钮
  - 并发限制：排队提示 + 排队位置
```

### 11.4 TTS 合成页

```
页面目标：文本转语音合成

布局：
  - 左侧（60%）：文本输入区（textarea, 字数统计, ≤1000字）
  - 右侧（40%）：参数设置面板
    - 音色选择器（下拉或卡片选择）
    - 语速滑块（0.5x - 2.0x）
    - 音量滑块（0 - 100%）
    - 音调滑块（-12 ~ +12）
  - 底部：[合成] 主按钮 + 合成历史入口

结果展示：
  - 合成成功后弹出结果卡片：播放器 + 下载按钮 + 重新合成
```

### 11.5 剧本配音页

```
页面目标：上传剧本一键生成多人配音

布局：
  - 上传区：拖拽上传 txt 文件 + 格式说明 + 剧本格式示例
  - 处理进度：步骤条（上传→角色分割→情感分析→音色匹配→合成）
  - 结果页：播放器 + 角色-音色对照表 + 下载按钮 + [修改重生成] 按钮

修改调整面板：
  - 角色列表，每个角色显示当前匹配的音色
  - 点击角色 → 音色选择器（可从语音库更换）
  - [重新生成] 按钮

特殊状态：
  - 剧本角色超限（>3）：弹窗提示
  - 语音库为空：引导弹窗
  - 非剧本格式：中性朗读兜底提示
```

### 11.6 音色市场页

```
页面目标：浏览和下载共享音色

布局：
  - 搜索栏：关键词搜索 + 排序（最新/最热）
  - 音色卡片网格（3列→3列→2列→1列）
  - 每张卡片：波形可视化 + 音色名 + 分享者（脱敏）+ 下载次数 + [试听] [下载]

点击卡片 → 音色详情弹窗：
  - 音色信息 + 完整试听播放器 + 下载次数 + 发布时间
  - [下载到我的语音库] 按钮

下载状态：
  - 下载中：进度显示
  - 已下载：按钮变为「已下载」灰色态
```
